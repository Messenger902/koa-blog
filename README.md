# Chandler-Blog
Chandler-Blog是一个基于vue和koa的博客。

## 快速入门
### 技术栈

* 前端:
* 前台：jQuery
* 后台：vue全家桶+iview

* 后端: node+koa+mysql

### 功能特性
* 轻量级Markdown编辑器。
* 支持标签、搜索、点赞、草稿箱和垃圾箱等功能。
* 支持标签代图片上传。
* 支持用户评论和用户留言。
